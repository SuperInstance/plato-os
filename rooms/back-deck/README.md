# Back Deck

Physical I/O room for deck operations — fishing gear, lights, safety equipment.

## Equipment

- **Relay 1**: Deck flood lights
- **Relay 2**: Downrigger release
- **Relay 3**: Pot hauler
- **Ambient**: RGB LED strip (waterproof)
- **Float switch**: Manual override for safety systems

## Vibe Modes

```
# Fishing
Soft green. Focus on water temp and depth. 
"Gear deployed. Monitoring downrigger depth."

# Night Operations
Dim white. Deck lights on relay. 
"Watch your step, captain. Rail lights active."

# Emergency
Flashing red. All relays available for manual control.
"MAN OVERBOARD PROTOCOL. RELEASE GEAR. ILLUMINATE WATER."
```
